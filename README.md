## アプリケーション概要  
ユーザーが登録したTaskでスケジュールの管理をすることができるTodoアプリケーションです。  
Taskの内容だけではなく、期限・進捗・コメントなどを追加できるようにし、  
Taskの管理を視覚的に容易にすることをコンセプトとしました。
